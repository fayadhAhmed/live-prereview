_Below is an example of how you can instruct your audience on installing and setting up your app. This template doesn't rely on any external dependencies or services._

   ```sh

   ```
2. add input upload file with id to get upload images from it
   ```sh
    <input type="file" name="imageUpload" id="id" multiple>
    example
    <input type="file" name="imageUpload" id="imageUpload" multiple>
   ```
* if do not want input accept multiple file delete multiple word it will be like this
   ```sh
    <input type="file" name="imageUpload" id="imageUpload">
   ```

3. at the above script file add script and add those varible varibles
   ```sh
    <script>

    /*
    *please fill thefields tomake tthe library works*
    uploadImageInput = upload file input tag it must be an id;
    livePreviewImageContainer = the container the image tag shoud be an id;
    divStyle = it array to style the div contan image how should an array be like ["className", "className",]
    mageStyle = it array to style the image how should an array be like ["className", "className",]
    */

        const uploadImageInput = document.getElementById("#id");
        let livePreviewImageContainer = document.getElementById("#id");
        divStyle = ["class", "class", "class"];
        imageStyle = ["class", "class", "class"];

       // example
        const uploadImageInput = document.getElementById("imageUpload");
        let livePreviewImageContainer = document.getElementById("livePreviewImageContainer");
        divStyle = ["col-2"];
   ```
